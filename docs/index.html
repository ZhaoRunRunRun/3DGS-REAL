3DGS REAL

跨平台3D高斯溅射建模工具 - 从照片到3D模型

GitHub仓库 下载应用

核心功能

📷

智能拍摄引导

实时相机预览，引导用户围绕物体旋转拍摄，自动计算覆盖角度，确保采集质量。

🖼️

相册批量导入

支持从相册批量选择照片，自动排序并映射环绕角度，快速开始建模流程。

✨

照片质量管理

自动评分系统，实时显示照片质量，支持过滤低质量图片，提高重建成功率。

🔄

本地/远端重建

灵活选择本地或远端重建模式，本地适合高性能设备，远端适合移动端和协作场景。

🎯

3DGS训练

集成COLMAP位姿恢复和3D Gaussian Splatting训练，自动化完成从照片到3D模型的全流程。

💾

多格式导出

支持导出SPLAT、PLY、GLB等多种格式，方便在不同平台和工具中使用。
